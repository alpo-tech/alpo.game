# CI/CD Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add GitHub Actions CI and automatic VPS deployment for `main`.

**Architecture:** CI validates Go code on pull requests and pushes. CD builds a
Linux binary, uploads it and static assets to a VPS over SSH, updates a
systemd-managed release symlink, restarts the service, and smoke-tests port
`8081`.

**Tech Stack:** Go, GitHub Actions, SSH/SCP, systemd, curl.
